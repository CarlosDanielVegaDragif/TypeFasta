# TypeFasta

TypeFasta is a competitive multiplayer typing game where players race against each other by typing words as quickly and accurately as possible.

This project focuses on delivering a solid real-time multiplayer experience using Steamworks, while providing a flexible foundation for future gameplay expansion.

---

## Features

### Core Gameplay

* Competitive typing matches for 2 to 7 players
* Fast-paced input system focused on speed and accuracy
* Designed for short, replayable multiplayer sessions

### Multiplayer (Steamworks)

* Lobby system powered by Steamworks
* Players can create and join lobbies
* Ability to invite friends through Steam once inside a lobby

### Word System

* Includes 100 total words:

  * 50 English words
  * 50 Spanish words
* Language selection available in the menu (English and Spanish)
* Spanish is the default language

### User Interface

* In-game input font updated for improved readability
* Simple and functional UI designed for clarity during matches

---

## Current State

This is an early version of the game that establishes the core systems:

* Multiplayer infrastructure
* Base gameplay loop
* Initial word database

The project is actively being developed and will continue to evolve.

---

## Known Limitations

* Limited word pool
* Only two supported languages (English and Spanish)
* Basic feature set with room for expansion
